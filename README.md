# 背單字前端 App

這是一個純前端的背單字軟體範例，使用 HTML、CSS、JavaScript 實作，沒有套用任何前端框架。

## 功能

- 卡片式單字背誦介面，可點擊翻面
- 顯示英文、翻譯、詞性、例句、字根分析
- 管理頁面可新增、更新、刪除單字
- 單字新增頁面提供「自動填入」按鈕，自動呼叫線上 API 填入欄位
- 資料保存在瀏覽器 LocalStorage

## 使用方式

1. 開啟 `index.html`
2. 在「管理單字」頁面新增或編輯單字
3. 點擊「單字卡片」查看單字並翻面

## API

- `dictionaryapi.dev`：取得詞性、例句、來源
- `MyMemory`：取得中文翻譯
